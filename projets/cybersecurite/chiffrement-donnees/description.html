Chiffrement et Déchiffrement des Données

AES-256 RSA-2048 OpenSSL PyCryptodome

Description

Implémentation d'algorithmes cryptographiques classiques et modernes.

Ce projet consiste à développer une application de chiffrement/déchiffrement de données en utilisant différentes techniques cryptographiques. L'objectif est de comprendre et d'appliquer les concepts de cryptographie pour sécuriser les communications et protéger les données sensibles.

1. Chiffrement Symétrique (AES) :

  • Le chiffrement symétrique utilise une seule clé pour le chiffrement et le déchiffrement des données.
  • L'algorithme AES est implémenté pour chiffrer des données à l'aide d'une clé de 128 bits.
  • Le programme propose une interface où l'utilisateur peut saisir un texte en clair, choisir une clé et voir le texte chiffré.
  • Les données chiffrées peuvent ensuite être déchiffrées à l'aide de la même clé.

2. Chiffrement Asymétrique (RSA) :

  • RSA est utilisé pour le chiffrement asymétrique, où une paire de clés publique/privée est générée.
  • Les utilisateurs peuvent envoyer un message chiffré à un destinataire en utilisant sa clé publique, et le destinataire peut le déchiffrer avec sa clé privée.
  • Le projet implémente également la signature numérique, permettant de signer un message et de le vérifier à l'aide de la clé publique.

Problème à résoudre

  • Implémenter le chiffrement symétrique (AES) et asymétrique (RSA)
  • Générer et gérer des paires de clés cryptographiques
  • Assurer l'intégrité des données via le hachage (SHA-256)
  • Simuler des communications sécurisées avec SSL/TLS

Méthodologie

1. Recherche théorique

Analyse des algorithmes AES, RSA, DES et étude des standards de sécurité modernes.

2. Développement des modules

Implémentation en Python avec PyCryptodome pour le chiffrement symétrique/asymétrique et utilisation d'OpenSSL pour simuler des échanges sécurisés.

3. Tests de validation

Vérification du chiffrement/déchiffrement et tests d'intégrité des données.

Outils Utilisés

Langage de programmation

Python

Algorithmes de chiffrement

AES (Advanced Encryption Standard), RSA (Rivest-Shamir-Adleman)

Outils

OpenSSL, PyCryptodome (bibliothèque Python)

Améliorations possibles

  • Implémentation d'autres algorithmes de chiffrement comme le chiffrement elliptique (ECC).
  • Service web sécurisé permettant à des utilisateurs externes de chiffrer/déchiffrer des messages via une interface web.
  • Ajout d'une authentification par certificat

Conclusion

Ce projet permet de comprendre l'importance du chiffrement dans la cybersécurité, en particulier pour assurer la confidentialité et l'intégrité des données échangées sur des réseaux non sécurisés. Il donne également un aperçu de l'implémentation pratique des différents algorithmes de cryptographie et de leur rôle dans la protection des informations.
