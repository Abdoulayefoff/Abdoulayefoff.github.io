Installation et Configuration d’un Serveur Web

Apache PHP MySQL

Description

Ce projet consiste à installer et configurer un serveur web Apache sur une machine Linux, en ajoutant le support de PHP et MySQL pour permettre l’exécution d’applications web dynamiques.

L’objectif est de créer un environnement de développement et d’hébergement web stable et performant.

Objectifs du projet

  • Installer et configurer un serveur Apache sous Linux
  • Ajouter le support de PHP pour le traitement des pages dynamiques
  • Installer et configurer MySQL pour la gestion des bases de données
  • Vérifier le bon fonctionnement du serveur avec un site web test
  • Optimiser et sécuriser les configurations pour de meilleures performances

Technologies et Outils Utilisés

  • Système d’exploitation : Linux (Ubuntu)
  • Serveur web : Apache
  • Langage backend : PHP
  • Base de données : MySQL/MariaDB
  • Outils complémentaires : Terminal Linux, commandes Shell

Étapes de réalisation

1. Installation du serveur Apache

Mise à jour des paquets et installation d’Apache.

2. Ajout du support PHP

Installation de PHP et vérification avec un fichier phpinfo().

3. Installation et configuration de MySQL

Installation de MySQL et sécurisation de l'installation.

4. Déploiement et test

Déploiement d’un site web simple en PHP et vérification via navigateur.

5. Optimisation et sécurisation

Configuration des permissions et activation du pare-feu.

Résultat et Perspectives

À l’issue du projet, le serveur web est fonctionnel, sécurisé et prêt à héberger des sites dynamiques. Il peut être amélioré avec :

  • L’activation du HTTPS via Let’s Encrypt
  • La mise en place d’une architecture multi-sites
  • L’ajout d’un système de logs et monitoring pour suivre les performances

Disponible pour une alternance de 12/24 mois
