Gestion de Bibliothèque

Vue.js Django JWT MySQL

Description

Application Web de gestion de bibliothèque avec interface moderne et sécurisée.

Cette application web permet la gestion d’une bibliothèque en facilitant l’ajout, la modification, la suppression et la consultation des livres. Les utilisateurs peuvent enregistrer de nouveaux livres avec leurs titres, auteurs, descriptions et catégories. L’application permet également d’afficher une liste de tous les livres disponibles, de rechercher un livre spécifique et de mettre à jour ses informations. Un système de suppression est inclus pour retirer des ouvrages. L’interface est conçue pour être intuitive et accessible, rendant la gestion du catalogue simple et efficace.

Installation


# Cloner le dépôt
git clone https://github.com/abdoulaye/gestion-bibliotheque.git

# Frontend
cd frontend
npm install
npm run serve

# Backend
cd backend
pip install -r requirements.txt
python manage.py runserver
