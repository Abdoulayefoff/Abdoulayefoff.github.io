Todolist en React.js avec GraphQL

React GraphQL Node.js JWT

Description

Application moderne de gestion de tâches avec système d'authentification sécurisé et gestion avancée des tâches.

Ce projet est une application de gestion de tâches (todolist) permettant aux utilisateurs de créer et organiser leurs listes de tâches de manière intuitive. L'objectif est de fournir une solution moderne et performante pour la gestion des tâches quotidiennes, en utilisant React.js pour l'interface utilisateur et GraphQL pour la gestion des requêtes côté serveur. Ce projet démontre mes compétences en développement full-stack avec une architecture moderne combinant React.js pour le frontend et GraphQL pour le backend.

Installation


#Prérequis
- Node.js installé

# Cloner le dépôt
git clone git@github.com:Abdoulayefoff/MyProjectsPortfolio.git

cd todolist-react-graphql

# Installer les dépendances
npm install

# Démarrer le serveur
npm run server

# Démarrer l'application
npm start
