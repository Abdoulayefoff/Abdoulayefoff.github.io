Introduction

Une application web de prise de notes simple et efficace, actuellement en développement avec Symfony.

NoteSync est un projet en cours qui vise à créer une application complète de gestion de notes. L'objectif est d'explorer les capacités de Symfony tout en développant une application utile au quotidien.

Projet en cours : Cette application est actuellement en développement actif. De nouvelles fonctionnalités sont régulièrement ajoutées et l'interface évolue constamment.

Entité Note

L'entité principale représentant une note dans l'application :

// src/Entity/Note.php
#[ORM\Entity]
class Note
{
  #[ORM\Id]
  #[ORM\GeneratedValue]
  #[ORM\Column]
  private ?int $id = null;

  #[ORM\Column(length: 255)]
  private string $title;

  #[ORM\Column(type: 'text')]
  private string $content;

  #[ORM\ManyToOne(targetEntity: User::class, inversedBy: 'notes')]
  private User $author;

  // Getters et setters... }

Contrôleur des Notes

Exemple de méthode pour créer une nouvelle note :

// src/Controller/NoteController.php
#[Route('/notes/new', name: 'note_new')]
public function new(Request $request): Response
{
  $note = new Note();
  $form = $this->createForm(NoteType::class, $note);

  $form->handleRequest($request);
  if ($form->isSubmitted() && $form->isValid()) {
    $note->setAuthor($this->getUser());
    $this->entityManager->persist($note);
    $this->entityManager->flush();

    return $this->redirectToRoute('note_index');
  }

  return $this->render('note/new.html.twig', [
    'form' => $form->createView()
  ]);
}
