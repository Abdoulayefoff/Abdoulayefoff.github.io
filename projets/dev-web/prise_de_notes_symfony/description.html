NoteSync - Application de Prise de Notes

Symfony Doctrine Twig Bootstrap MySQL

Introduction

Une application web de prise de notes simple et efficace, développée avec Symfony.

NoteSync permet aux utilisateurs de créer, organiser et gérer leurs notes en ligne avec une interface intuitive. Ce projet a été réalisé pour explorer les fondamentaux de Symfony tout en créant une application utile au quotidien.

Entités Principales

User

// src/Entity/User.php
#[ORM\Entity]
class User implements UserInterface
{
  #[ORM\Id]
  #[ORM\GeneratedValue]
  #[ORM\Column]
  private ?int $id = null;

  #[ORM\Column(length: 180, unique: true)]
  private string $email;

  #[ORM\Column]
  private string $password;

  #[ORM\OneToMany(mappedBy: 'user', targetEntity: Note::class)]
  private Collection $notes;
}

Note

// src/Entity/Note.php
#[ORM\Entity]
#[ORM\HasLifecycleCallbacks]
class Note
{
  #[ORM\Id]
  #[ORM\GeneratedValue]
  #[ORM\Column]
  private ?int $id = null;

  #[ORM\Column(length: 255)]
  private string $title;

  #[ORM\Column(type: 'text')]
  private string $content;

  #[ORM\Column(type: 'datetime')]
  private \DateTimeInterface $createdAt;

  #[ORM\ManyToOne(inversedBy: 'notes')]
  #[ORM\JoinColumn(nullable: false)]
  private User $user;

  #[ORM\PrePersist]
  public function setCreatedAtValue(): void
  {
    $this->createdAt = new \DateTimeImmutable();
  }
}

Extraits de Code

1. Contrôleur de Notes

// src/Controller/NoteController.php
#[Route('/notes', name: 'app_note_')]
class NoteController extends AbstractController
{
  #[Route('/', name: 'index', methods: ['GET'])]
  public function index(NoteRepository $noteRepo): Response
  {
    return $this->render('note/index.html.twig', [
      'notes' => $noteRepo->findByUser($this->getUser())
    ]);
  }

  #[Route('/new', name: 'new', methods: ['GET', 'POST'])]
  public function new(Request $request, EntityManagerInterface $em): Response
  {
    $note = new Note();
    $note->setUser($this->getUser());

    $form = $this->createForm(NoteType::class, $note);
    $form->handleRequest($request);

    if ($form->isSubmitted() && $form->isValid()) {
      $em->persist($note);
      $em->flush();

      return $this->redirectToRoute('app_note_index');
    }

    return $this->render('note/new.html.twig', [
      'form' => $form->createView()
    ]);
  }
}

2. Repository Personnalisé

// src/Repository/NoteRepository.php
class NoteRepository extends ServiceEntityRepository
{
  public function findByUser(User $user): array
  {
    return $this->createQueryBuilder('n')
      ->andWhere('n.user = :user')
      ->setParameter('user', $user)
      ->orderBy('n.createdAt', 'DESC')
      ->getQuery()
      ->getResult();
  }
}

Installation Locale

Prérequis

  • PHP 8.2+
  • Composer
  • MySQL 5.7+
  • Node.js (optionnel pour les assets)

Commandes Clés

$ git clone https://github.com/votrecompte/notesync.git
$ cd notesync
$ composer install
$ npm install # Optionnel
$ symfony console doctrine:database:create
$ symfony console doctrine:migrations:migrate
$ symfony console doctrine:fixtures:load # Données de test
$ symfony serve -d

L'application sera accessible à l'adresse http://localhost:8000
