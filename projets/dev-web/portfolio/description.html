Portfolio sur GitHub Pages

HTML5 CSS Bootstrap JavaScript GitHub Pages

Description

Portfolio professionnel hébergé sur GitHub Pages pour présenter mes projets et compétences.

Ce projet consiste en la création d’un portfolio en ligne pour présenter mes projets en développement web, cybersécurité, développement logiciel et Administration système et réseaux. Il est hébergé sur GitHub Pages et permet aux visiteurs d'explorer mes travaux et d’accéder directement aux codes sources via GitHub.
